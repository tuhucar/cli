# TuhuCar for OpenCode

## Installation

1. Install the CLI:
   ```bash
   npm install -g @tuhucar/cli
   ```

2. Initialize configuration:
   ```bash
   tuhucar config init
   ```

3. Install skills:
   ```bash
   tuhucar skill install
   ```

This will register tuhucar as a plugin in `~/.opencode/opencode.json`.

## Usage

Once installed, ask your AI assistant car maintenance questions like:
- "我的2024款朗逸多久换机油？"
- "大众帕萨特轮胎气压多少？"
