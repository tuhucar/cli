# TuhuCar for Codex

## Installation

1. Install the CLI:
   ```bash
   npm install -g @tuhucar/cli
   ```

2. Initialize configuration:
   ```bash
   tuhucar config init
   ```

3. Install skills:
   ```bash
   tuhucar skill install
   ```

This will create a symlink at `$CODEX_HOME/skills/tuhucar` pointing to the tuhucar skill definitions.

## Usage

Once installed, ask your AI assistant car maintenance questions like:
- "我的2024款朗逸多久换机油？"
- "大众帕萨特轮胎气压多少？"
